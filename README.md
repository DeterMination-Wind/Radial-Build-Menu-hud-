# 圆盘快捷建造 / Radial Build Menu

按住热键在鼠标周围打开圆盘 HUD，移动鼠标到图标上后松开热键即可快速切换建造建筑。

Hold a hotkey to open a radial HUD around your cursor. Move onto an icon and release the hotkey to quickly switch the selected build block.

## 功能 / Features

- 8 个可配置槽位 / 8 configurable slots
- “设置 → 控制”里可绑定热键 / Hotkey is configurable in Settings → Controls
- “设置 → 圆盘快捷建造”可配置槽位、HUD 缩放与透明度 / Slots + HUD scale/opacity are configurable in Settings → Radial Build Menu
- 规则切换：按地图时长/星球切换到另一套槽位 / Rule switching: switch to another slot profile by map time or planet

## 安装 / Install

将 `radial-build-menu.zip` 放入 Mindustry 的 `mods` 目录即可。

Put `radial-build-menu.zip` into Mindustry's `mods` folder.

## 构建（在本 Mindustry 源码工程内）/ Build (inside this Mindustry source checkout)

在 `Mindustry-master` 根目录运行：

From the `Mindustry-master` root directory:

`./gradlew.bat :radial-build-menu:jar`

输出文件：

Output:

`mods/radial-build-menu/build/libs/radial-build-menu.zip`
